先从第一层说起：日本白皮书的措辞升级，本质上是“身份定位”的重塑。2024年之前，日本对中国的描述是“关切”和“警惕”，2025年直接跳到“最大战略挑战”。这不是简单的修辞变化，而是为政策突破铺路。数据支撑：日本2023-2027年防卫费计划从GDP的1%提升至2%，这意味着年度军费将从约5万亿日元增至10万亿日元（约670亿美元），超过俄罗斯2024年军费预算。而2027年这个时间节点，恰好与日本列装的“反击能力”装备形成周期——包括400枚美制战斧巡航导弹（射程1600公里）和国产高超音速武器（预计2026年试射）。

## 四、技术说明

### Style Routing

所有风格通过 `STYLE_ROUTER` 字典实现 O(1) 路由：

```python
STYLE_ROUTER = {
    ContentStyle.MILITARY:         (SYSTEM_PROMPT_MILITARY,         MILITARY_TOUTIE_PROMPT,    0.7),
    ContentStyle.STORY_NARRATIVE:  (SYSTEM_PROMPT_STORY_NARRATIVE,  STORY_NARRATIVE_PROMPT,    0.85),
    ContentStyle.SHARP_COMMENTARY: (SYSTEM_PROMPT_SHARP_COMMENTARY, SHARP_COMMENTARY_PROMPT,   0.6),
    ContentStyle.DATA_LIST:        (SYSTEM_PROMPT_DATA_LIST,        DATA_LIST_PROMPT,          0.5),
    ContentStyle.FLASH_NEWS:       (SYSTEM_PROMPT_FLASH_NEWS,       FLASH_NEWS_PROMPT,         0.5),
    ContentStyle.DISCUSSION:       (SYSTEM_PROMPT_DISCUSSION,       DISCUSSION_PROMPT,         0.7),
}
```

### 共享红线

所有军事类风格的 System Prompt 末尾统一追加 `MILITARY_RED_LINES`，
包含「军事真实性红线」和「国家立场红线」，确保各风格均遵守核心底线。

### Pipeline 调用

在 `pipeline.py` 中修改 `content_style` 参数即可切换风格：

```python
"content_style": "military"           # 👈 改成对应值
# 可选值: military / story_narrative / sharp_commentary / data_list / flash_news / discussion
```
